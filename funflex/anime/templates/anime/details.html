Project Overview

FunFlex is an anime streaming web application built using Django. It allows users to register, log in, browse anime, and add favorites. Admins can manage anime titles, categories, and episodes.

Key Features

  • User Authentication (Register/Login)
  • Anime Categories & Search
  • Favorites List
  • Admin Panel for Uploading Anime
  • Dark-Themed UI with Netflix-Style Effects
